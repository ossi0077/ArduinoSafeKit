# Gongzipsa 금고 키트 제작
키패드와 서보모터를 이용한 금고 키트

## 사용 부품
- 아두이노 나노
- 서보모터 SG90 1개
- 3*4 키패드 1개

## 라이브러리
- Keypad by Mark Stanley, Alexander Brevig
